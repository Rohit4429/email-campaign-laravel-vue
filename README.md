## Email Compaign Send Bulk User (Laravel and Vuejs)

## Setup

- composer install
- npm install

setup your databse 
- php artisan migrate
- php artisan db:seed

## Login and Uses
- Email :- admin@gmail.com
- password :- password

After login, click on the campaign menu and then upload the file. Only name and email should be kept in the CSV file.
